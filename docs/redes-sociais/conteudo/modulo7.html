Dia 1-3: Nascimento (IDEAL)

Todos fazendo. Alcance menor. So vale se tiver angulo muito diferente.

7.9

Replicando Viralidade

Framework: Viral Video Breakdown

1. Hook (0-3s) - O que parou o scroll? Texto? Visual? Audio? Replique o padrao.
2. Retencao (3-15s) - Como manteve atencao? Ritmo? Cortes? Musica? Liste os elementos.
3. Emocao - Qual sentimento gerou? Raiva? Admiracao? Humor? Use mesma emocao em novo contexto.
4. CTA - Qual acao pediu? Comentar? Compartilhar? Seguir? Adapte para seu nicho.
5. Formato - Estrutura narrativa? POV? Lista? Before/After? Clone a estrutura.
Exercicio Pratico

Salve 20 videos virais do seu nicho. Quebre cada um com framework acima. Identifique padroes. Crie template proprio.
