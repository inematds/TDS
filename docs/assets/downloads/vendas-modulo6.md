## 6.2 Prompts que Vendem

### Framework CRISP para Prompts

| Letra | Significado | Exemplo |
|-------|-------------|---------|
| C | Contexto | "Sou um consultor de marketing digital..." |
| R | Role (Papel) | "Atue como um copywriter experiente..." |
| I | Instrucao | "Crie 5 opcoes de assunto de email..." |
| S | Especificacoes | "Maximo 50 caracteres, tom informal..." |
| P | Padrao/Exemplo | "No formato: [Assunto] - [Por que funciona]" |

### 6 Prompts Prontos para Vendas

**1. Pesquisar Prospect**
```
Analise o perfil LinkedIn de [NOME] na empresa [EMPRESA].
Identifique: 1) Possiveis dores, 2) Pontos de conexao,
3) Conquistas recentes. Meu produto e [PRODUTO].
```

**2. Email de Prospeccao**
```
Atue como SDR especialista em cold email.
Crie email para [CARGO] de empresas de [SETOR].
Max 100 palavras, foca em [DOR], nao parecer automatizado,
terminar com pergunta simples.
```

**3. Script de Call**
```
Crie roteiro de ligacao de 3 min para [PRODUTO].
Estrutura: 1) Abertura com permissao, 2) Qualificacao,
3) Pitch 30seg, 4) Objecao "nao tenho tempo", 5) CTA.
```

**4. Responder Objecao**
```
Prospect disse: "[OBJECAO]". Contexto: [CONTEXTO].
Crie 3 respostas: 1) Questionamento, 2) Reframe,
3) Prova social. Max 2 frases cada.
```

**5. Proposta Comercial**
```
Crie proposta para [CLIENTE] que precisa de [SERVICO].
Inclua: resumo executivo, problema, solucao, beneficios,
investimento com opcoes, proximos passos, garantias.
```

**6. Sequencia Follow-up**
```
Crie 5 emails de follow-up para leads interessados
que nao compraram. Produto: [PRODUTO].
Cada email: objetivo diferente, < 80 palavras,
CTA claro. Espacamento: D+1, D+3, D+7, D+14, D+30.
```

### Dica: Refinamento Iterativo
Nunca aceite a primeira resposta. Refine com:
- "Torne mais conversacional/formal"
- "Reduza para metade do tamanho"
- "Adicione mais urgencia"
- "Faca 3 versoes diferentes"

## 6.3 Automacao de Vendas

### 5 Automacoes Essenciais

| Automacao | Descricao | Fluxo |
|-----------|-----------|-------|
| Lead → CRM | Cria automaticamente no CRM | Formulario → Zapier → CRM → Email |
| Nurturing | Emails automaticos de valor | Lead → Email D+1 → D+3 → D+7 |
| Lembrete Follow-up | Alerta quando lead nao responde | Email + 3 dias → Alerta Slack |
| Agendamento | Link que sincroniza com calendario | Calendly → Google → Lembrete |
| Pos-Venda | Onboarding automatico | Compra → Email → D+7 check-in |

### Chatbot de Vendas no WhatsApp

**Fluxo de Conversacao:**
1. BOT: "Oi! Como posso te ajudar?"
2. Botoes: [Conhecer servicos] [Falar com consultor]
3. BOT: "Qual seu maior desafio com [area]?"
4. Qualificacao: problema, urgencia, orcamento
5. BOT: "Vou te conectar com especialista. Qual horario?"

**Dica:** Sempre tenha opcao de falar com humano.
